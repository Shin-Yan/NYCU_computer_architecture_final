# Computer Architecture final project

Install: 
```
$ git clone https://github.com/Shin-Yan/NYCU_computer_architecture_final.git
```
Compile:
```
// In developing mode, use make
$ make

// In Virtual Machine to test the code, use make vm
$ make vm
```

Execution:
```
// In developing mode
$ ./cpu_version
$ ./gpu_version

// In Virtual Machine
$ ./cpu_vm
$ ./gpu_vm
```
